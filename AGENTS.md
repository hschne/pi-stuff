# Agent Guidelines for pi-stuff

## About

Personal Pi coding agent configuration — extensions, skills, and settings for the [Pi coding agent](https://shittycodingagent.ai).

## Issue Tracker & PRDs

PRDs and issues are stored locally as Markdown files, **not** in GitHub Issues.

- PRD: `.agents/PRD.md`
- Issues:
  - Numbered by priority.
  - Open: `.agents/issues/<ID>-<slug>.md`
  - Done: `.agents/issues/done/<ID>-<slug>.md`
- PRDs follow the template in the `prd` skill.
- Implementation issues reference their parent PRD.
